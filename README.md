## Getting started
Play around by running the model locally.

### Prerequisites

- Python 3.10+ 
- pip
- PyTorch with CUDA

```bash
# check versions
python --version
pip --version
```

### Installation
1. Clone the repo
```bash
git clone https://github.com/sidsurakanti/cifar-10.git
cd /path/to/project/
```
2. Install dependencies
```bash
pip install -r requirements.txt
```
